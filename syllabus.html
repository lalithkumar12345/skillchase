Advance Cyber Security Course Syllabus

Module 1 - Basics of Linux

Assuming you have little to no experience with Linux, this session will cover the basics of what you need to know to get started. We will briefly touch on topics such as what Linux is, why it is important, where you can download it, and how to install it. By the end of this session, you should have a better understanding of Linux and be able to take your first steps in using it.

Module 2 - Introduction to Ethical Hacking

Introduction to ethical hacking techniques, understanding threats, and security vulnerabilities.

Module 3 - Footprinting and Reconnaissance

Learn how to gather information and map out networks for security assessments.

Module 4 - Scanning Networks

Understanding network scanning tools and techniques for vulnerability identification.

Module 5 - Enumeration

Dive into enumeration techniques, focusing on extracting information from systems and networks.
